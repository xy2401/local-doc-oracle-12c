Description of the illustration em_red_format_edit.png

Image em_red_format_edit.png shows the Edit format dialog box, with the following components, from top to bottom:

  • The Format Name field lists the name of the shortcut format.

  • The Description field lists a brief description of the format.

  • The Sensitive Column Type menu lists the available sensitive column types for this shortcut format.

  • The Redaction Function menu lists the available redaction functions, such as FULL or RANDOM. Based on your selection, a label with a description of the redaction function follows.
