Description of the illustration thematic_map_display.gif

This figure illustrates a map in which the text style used for the county name label for each of several adjacent California counties reflects the total population of the county. In this illustration, several county name labels have different text styles associated with them. Also, some counties have no name label because the <label> element of the theme's styling rules specifies the advanced style as its label style, as explained in text after the illustration.
