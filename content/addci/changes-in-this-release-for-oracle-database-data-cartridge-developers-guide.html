Changes in Oracle Database 12c Release 2 (12.2.01)

The following are changes in Oracle Database Data Cartridge Developer's Guide for Oracle Database 12c release 2 (12.2.0.1).

  • Support for converting BFILE type to binary large objects (BLOB).

  • Support for local domain index on reference partitioned and composite partitioned tables has been introduced. For more information, refer to Partitioned Domain Indexes.
