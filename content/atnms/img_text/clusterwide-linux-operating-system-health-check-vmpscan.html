Clusterwide Linux Operating System Health Check (VMPScan)

Shows the Clusterwide Linux Operating System Health Check (VMPScan) section of the HTML Report output.

Shows HostView table with list of 3 hosts, each with results which can be clicked including Health(1), Error(0) and warning(10).

Underneath shows ClusterView (Key Parameters) Clusterview Report with list of parameters broken down by net, os, and storage.
