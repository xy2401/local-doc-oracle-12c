Shows an architecture diagram for Oracle Trace File Analyzer.

The command line tool tfactl is used, which connects to the Oracle Trace File Analyzer daemon, which invokes scripts and collects alerts and log files locally and also communicates with remove Oracle Trace File Analyzer daemons on each of the other nodes in the cluster.

Each remote Oracle Trace File Analyzer daemon also invokes scripts and collects alerts and log files, then all area returned to the originating node with a cluster wide collection.
