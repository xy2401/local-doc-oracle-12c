This image depicts a flow of events that occur for on-demand collections.

  • A DBA or sysadmin requests a summary with tfactl summary and Oracle Trace File Analyzer provides the summary of systems, changes, events, health and analysis

  • If there are any problems and more information is required, then the DBA requests further analysis using tfactl and Oracle Trace File Analyzer performs analysis using support tools where required

  • If more help is still required, then the DBA requests a collection using tfactl diagcollect and Oracle Trace File Analyzer collects and packages all relevant diagnostics

  • The DBA then uploads the package to Oracle Support for help resolving

  • The sections marked request further analysis using tfactl and performance analysis using support tools where necessary are highlighted to indicate this is the piece of the flow being focused on.
