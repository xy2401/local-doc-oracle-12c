This image depicts a flow of events that occur for automatic collections.

A significant problem occurs in the Oracle Grid Infrastructure or Databases.

Oracle Trace File Analyzer Collector automatically detects the event then collects and packages relevant diagnostics and notifies the relevant DBA and or Sys Admin by email.

The DBA(s) or Sys Admin(s) can then upload the collection to Oracle Support for further help.
