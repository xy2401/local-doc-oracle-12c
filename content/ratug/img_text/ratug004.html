Description of the illustration ratug004.png

This diagram shows how to set up the test system to consolidate three workload captures. There are two main parts in this diagram.

The upper part of the diagram consists of a dotted box labeled Production System. It contains three databases and three captures, one capture for each database. An arrow points down from the Production System part to the lower part of the diagram.

The lower part of the diagram consists of a dotted box labeled Test System. It contains a dotted box labeled Consolidated Capture, and a large database labeled CDB. The Consolidated Capture dotted box contains a label named New Capture Directory. Under New Capture Directory are three labels representing each of the three captures. The Consolidated Capture dotted box is connected to the CDB database. The CDB database contains three smaller PDB databases, one for each database in the Production System part of the diagram.

End of description.
