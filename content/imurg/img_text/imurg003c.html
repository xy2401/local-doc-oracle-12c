Description of the illustration imurg0003c.png

This is a text description of imurg003c.

At the top: The double-sided arrows show that multimedia content in the database can be accessed with JDBC and OCI.

In the center: Using Oracle Multimedia, Oracle Database holds rich content in tables along with traditional data. Rich content includes audio, image, video, and DICOM content, such as X-rays. A server-side media parser and an image processor run inside the embedded Java Virtual Machine (JVM). The media parser supports format and application metadata parsing, and can be extended to support additional formats. The image processor provides image processing for operations such as producing thumbnail-size images, converting image formats, and image watermarking.

On the right: Using Oracle Multimedia procedures or methods enables import and export operations between the database and external file storage systems. The double-sided arrow connecting Oracle Database with External File Storage shows this data movement.
