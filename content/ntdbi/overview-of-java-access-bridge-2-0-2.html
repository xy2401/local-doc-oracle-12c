Overview of Java Access Bridge 2.0.2

Java Access Bridge 2.0.2 enables assistive technologies to read Java applications running on the Windows platform.

Assistive technologies can read Java-based interfaces, such as Oracle Universal Installer and Oracle Enterprise Manager Database Express.

For a list of supported system configurations, including supported versions of Microsoft Windows and Java SE, see section "Supported System Configuration" available at the following link location:

http://docs.oracle.com

During installation, Oracle Universal Installer uses the Java Runtime Environment (JRE) 1.8 contained in an Oracle Database installation media. The JRE enables the use of Java Access Bridge during installation.
