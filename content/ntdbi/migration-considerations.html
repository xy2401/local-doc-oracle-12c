Migration Considerations

Review the migration considerations in this topic.

You can migrate an existing Oracle Database 11g for 32-bit Windows to Oracle Database 12c for 64-bit Windows.

Oracle Database Upgrade Assistant has an option, Move Database from a Different Release 12.1 Oracle home, which lets you move your database from a Windows Built-in Account secured home to a Windows User Account secured home.
