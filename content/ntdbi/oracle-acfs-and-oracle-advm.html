About Oracle ACFS and Oracle ADVM

Oracle ACFS extends Oracle ASM technology to support all of your application data in both single instance and cluster configurations.

Oracle ADVM provides volume management services and a standard disk device driver interface to clients. Oracle Automatic Storage Management Cluster File System communicates with Oracle ASM through the Oracle Automatic Storage Management Dynamic Volume Manager interface.

Oracle ACFS and Oracle ADVM Support on Windows

Oracle ACFS and Oracle ADVM are supported on Windows Server 2008 R2 x64, Windows Server 2012 x64, and Windows Server 2012 R2 x64.
