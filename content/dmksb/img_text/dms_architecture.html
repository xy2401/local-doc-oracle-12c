The DMS architecture diagram provides an overview of the architecture used in Oracle Data Masking and Subsetting. The left portion of the architecture diagram shows the Oracle Cloud Control UI and EMCLI Command Line interface that can be used to access Oracle Data Masking and Subsetting. The middle portion of the diagram shows the data masking and subsetting components that are part of the Oracle Enterprise Manger repository, and the right portion of the diagram shows the Oracle Enterprise Manager Agent and gateways.
