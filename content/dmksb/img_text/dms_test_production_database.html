This figure shows a test/development database and a production database used by Oracle Data Masking and Subsetting. The sensitive production data is masked and subsetted and is shown appearing in the test/development database. Also, the following benefits of using Oracle Data Masking and Susbetting are listed:

  • discovering sensitive data

  • modeling application data

  • masking data using format library

  • subseting based on goal/condition

  • masking/subsetting in export or on staging

  • masking in workload captures and clone database
