This is a screenshot of creating an object rule for a data subsetting definition process that appears when you try to edit an ADM. On this page, you can create an object rule by selecting an application for which you want to provide a rule. You can then choose to select all rows, or some rows by specifying a percentage portion of the rows. For finer granularity, you can also specify a Where clause.
