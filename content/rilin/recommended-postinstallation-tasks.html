4.2 Recommended Postinstallation Tasks

Oracle recommends that you complete these tasks after installation.

  • Setting Up Additional User Accounts
    You can set up additional user accounts to manage your database.
  • Downloading and Installing the ORAchk Health Check Tool
    Download and install the ORAchk utility to perform proactive heath checks for the Oracle software stack.
