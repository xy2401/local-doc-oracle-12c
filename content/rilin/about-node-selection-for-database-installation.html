2.2.5 About Cluster Node Selection for Database Installation

Select the cluster nodes to install Oracle RAC or Oracle RAC One Node databases.

You can use Leaf Nodes to host Oracle RAC database instances that run in read-only mode, which become Oracle RAC Reader Nodes. Oracle RAC Reader Nodes facilitate Oracle Flex Cluster architecture by allocating a set of read/write instances running Online Transaction Processing (OLTP) workloads and a set of read-only database instances across Hub Nodes and Leaf Nodes in the cluster.

When the cluster is started, the Oracle Database instance on the Leaf Node automatically becomes an Oracle RAC Reader Node and the Hub Node instance remains a typical node.
