Description of the property graph architecture illustration

This figure shows the Oracle Property Graph Architecture receiving Java API input from the Graph Analytics layer and the Graph Data Access layer. These layers also communicate with each other and with the REST Web Services.

The Graph Analytics layer consists of in-memory parallel graph analytics.

The Graph Data Access layer includes the Blueprints and Text Search APIs.

The scalable and persistent storage (Oracle Database) includes PL/SQL packages, PG Schema (VT$, GE$, IT$, SS$, GT$), SQL-based graph query, SQL-based graph analytics, OLS Security, Compression, and Oracle Text Index. Oracle Database communicates with the Grapg Data Access Layer.
