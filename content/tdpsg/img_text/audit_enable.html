Description of the illustration audit_enable.png

Image audit_enable.png shows the Enable Audit Policy dialog. From top to bottom, left to right, are the following components:

At the top is the label Enable Audit Policy.

Following this label are the instructions, "Enabling of an object-level Unified Audit Policy takes effect immediately. Other Unified Audit Policy types take effect on subsequent user sessions."

Following the instructions is the Few Users option for Enable Policy selected.

Following this check box are the Add User button, Remove User button, and Except below users check box (which enables you to exclude specific users from the audit policy).

Following these buttons is a table with the columns User, Success, and Failure.

Following these buttons at the lower right corner are the OK and Cancel buttons.
