Description of the illustration dv_realm_object.gif

Graphic dv_realm_object.gif shows the Realm secured objects page. At the top, from left to right is General, then Realm secured objects, then Realm authorizations, and then Review, which indicates your progress. From top to bottom are the following components:

  • The label Create Realm : Realm secured objects. To the right of this label are the Back button, a label saying Step 2 of 4, the Next button, the Done button, and the Cancel button.

  • The text "Define a database schema or database role that is protected by the realm."

  • A table listing the components of the realm: Owner, Object Name, and Object Type. Preceding this table are from left to right, the buttons View, Add, and Remove.
