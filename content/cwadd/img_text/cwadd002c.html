Description of the illustration cwadd002.eps

This figure illustrates an Oracle Flex Cluster, a multinode, clustered architecture made up of Hub Nodes and Leaf Nodes, where both applications and the Oracle Database are managed using Oracle Clusterware for high availability. In the illustration, there are three Hub Nodes and six Leaf Nodes. There are two Leaf Nodes connected to each Hub Node, and the Hub Nodes are connected to a pool of storage, which lies outside the cluster. Each of the three nodes exchanges heartbeats with the other two.

The cluster connects to six disks that constitute the storage pool.

End of the description.
