Description of the illustration adxdk119.gif

An icon of a user at a computer, labeled “User”, is connected by a bidirectional arrow to a browser icon, labeled “Browser”. An arrow leads to the browser icon from a document icon labeled “Query result transformed by XSL stylesheet”. An arrow leads from the browser icon to a server icon, labeled “Java Web Server”. An arrow leads from the server icon to a document icon labeled “XML-formatted SQL queries”. An arrow leads from that SQL-queries document icon to a box labeled “Servlet running in Oracle Database 10g”. Inside this servlet box is the label “XSQL Servlet” and two smaller boxes, labeled “XML parser with XSLT processor” and “XML SQL utility”. Two dotted lines lead from the servlet box to a cylinder icon labeled “Database”, indicating that the servlet is inside the database.
