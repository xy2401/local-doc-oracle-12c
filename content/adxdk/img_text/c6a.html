Description of the illustration c6a.gif

A window shows the "Include Directories" window. In a text field, the include path described in the text is entered.

Above this is the tool bar which contains menu icons to Add New Line, Cancel, and up or down arrows for moving from one existing Include Directory to another. At the bottom of the window are OK and Cancel buttons and a Macros button, leading to lists of Macros.
