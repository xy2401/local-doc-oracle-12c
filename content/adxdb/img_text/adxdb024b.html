Description of the illustration adxdb024.eps

A cylinder labeled Oracle Database contains a dotted rectangle labeled Oracle XML DB, which contains a rectangle labeled XMLType Tables, Columns, Views. An arrow points from that cylinder to a box labeled JDBC. An arrow points from that box to a box labeled Oracle XML DB XMLType Instance. An arrow points from that box to a box labeled XML DOM Tree. An arrow also points from a box labeled Java DOM API to the box labeled XML DOM Tree. An arrow points from the box labeled XML DOM Tree to another box labeled JDBC. An arrow points from that JDBC box to the cylinder. This last arrow has two labels: (1) Changed XML Data and (2) Saved back in the Database.
