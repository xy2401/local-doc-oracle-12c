Installing ODBC Drivers for Oracle Solaris

If you intend to use ODBC, then install the most recent ODBC Driver Manager for Oracle Solaris.

Download and install the ODBC Driver Manager from the following website:

http://www.unixodbc.org

Review the minimum supported ODBC driver releases, and install ODBC drivers of the following or later releases for all Oracle Solaris distributions:

unixODBC-2.3.1 or later
