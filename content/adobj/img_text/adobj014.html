Description of the illustration adobj014.gif

The figure shows the people_reltab relational table at top left, with lines connecting the table's employee column, which includes the name and address objects as attributes, and the nested table of phone numbers, phones_ntab.

The figure is a graphical representation of the CREATE TABLE people_reltab statement that precedes the figure in the text.
