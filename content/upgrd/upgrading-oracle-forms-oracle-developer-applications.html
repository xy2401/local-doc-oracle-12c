About Upgrading Oracle Forms or Oracle Developer Applications

Oracle Forms includes a number of new features.

Forms applications run the same on Oracle9i, Oracle Database 10g, and Oracle Database 11g. In Oracle Database 12c, Oracle Database Development Guide is renamed to Oracle Database Advanced Application Developer's Guide. Review the new features and changes described for developing applications, and for the new Oracle Database release.

Oracle Database New Features Guide to determine if any of the features of Oracle Database 12c can benefit your applications, or otherwise affect them
