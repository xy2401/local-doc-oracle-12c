Description of the illustration haovw_bronze.gif

The graphic shows a single instance HA architecture. There are two boxes and a bulleted list. The box on the left contains a graphic depiction of a single instance database including icons VMs, Multitenant, Dedicated Databases, and Backups, with an arrow pointing to the box on the right. There is no relationship drawn between any of the icons. The right box contains icons meant to depict offiste backups including the cloud and a tape drive server. The bulleted list is as follows:

  • HA capabilities included with single-instance Oracle Database

  • Consolidation options:

    • VM for dedication resource

    • Oracle Multitenant for lowest cost

    • Multiple databases on a single physical machine

  • Unrecoverable outage - Restore from backup

  • Disaster Recovery (DR) - Offsite cloud backup or tape
