Description of the illustration haovw_gold.gif

The graphic shows a gold tier HA architecture. There are two boxes and a bulleted list. The box on the left is labeled "Site A" contains icons for HA Cluster, Multitenant, Dedicated Databases, and Backups, with an arrow pointing to the box on the right. The arrow has a label readin "Database Replication sync/async." There is no relationship drawn between any of the icons. The right box is labeled "Site B" and contains icons for HA Cluster, Multitenant, Dedicated Databases, Backups, and a tape drive server. The bulleted list is as follows:

  • Real-time data protection, HA and DR using Active Data Guard:

    • Best corruption protection

    • Zero or near-zero data loss

    • Automatic database failover

    • Offload read-only and backups

    • Database rolling maintenance

  • Flexible logical replication using Oracle GoldenGate - Replica open read-write

  • Coordinated site failover using Oracle Site Guard
