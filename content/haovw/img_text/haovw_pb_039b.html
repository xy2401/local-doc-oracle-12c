Description of the illustration haovw_platinum.gif

The graphic shows a platinum tier HA architecture. There are two boxes, a cloud above the boxes, and a bulleted list. The box on the left is labeled "Site A" contains icons for Backups, Multitenant, Tape, and Bi-directional Replication between databases, with an arrow pointing to the box on the right. The arrow has a label reading "Database Replication sync." There is no relationship drawn between any of the icons. The right box is labeled "Site B" and contains icons for Multitenant, Dedicated Databases, Backups, and Tape. The cloud over the two boxes has text reading "Global Data Services" and "Application Continuity The bulleted list is as follows:

  • Outages masked from applications, in-flight transactions preserved - Application Continuity

  • Zero data loss failover, LAN or WAN - Active Data Guard/Far Synch

  • Bi-directional replication and zero downtime maintenance - Oracle GoldenGate

  • Online patching for applications - Edition-based Redefinition

  • Automated workload management - Oracle Global Data Services
