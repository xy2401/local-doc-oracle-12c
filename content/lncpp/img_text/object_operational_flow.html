Description of the illustration object_operational_flow.gif

This illustration describes basic object operational flow.

  1. Initialize OCCI in object mode

  2. Pin the object or bring the object into the client-side cache

  3. Operate on the object in the cache

  4. Mark the object as dirty

  5. Flush changes to the object

  6. Refresh the object
