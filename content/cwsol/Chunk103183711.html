B.4.6 Using the Correct LSNRCTL Commands

To administer Oracle Database 12c Release 2 local and scan listeners using the lsnrctl command, set your $ORACLE_HOME environment variable to the path for the Oracle Grid Infrastructure home (Grid home).

Do not attempt to use the lsnrctl commands from Oracle home locations for previous releases, as they cannot be used with the new release.
