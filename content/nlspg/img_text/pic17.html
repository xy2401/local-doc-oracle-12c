Description of the illustration

This image shows the Existing Definitions dialog box. It contains scrolling menus for Language(ID), Language Abbreviation, and Territory(ID) in the top row. In this image, AMERICAN(1) is highlighted. The second row of scrolling menus contains Territory Abbreviation, Chracter Set(ID), and Linguistic Sort(ID). Open and Close buttons appear in the lower right of the dialog box. Text at the bottom of the dialog box shows that the Corresponding File Name for AMERICAN(1) is lx00001.nlb.
