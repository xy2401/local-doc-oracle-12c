6.314 USER_ANALYTIC_VIEWS

USER_ANALYTIC_VIEWS describes the analytic views owned by the current user.

Its columns (except for OWNER) are the same as those in ALL_ANALYTIC_VIEWS.

Note:

This view is available starting with Oracle Database 12c Release 2 (12.2.0.1).
