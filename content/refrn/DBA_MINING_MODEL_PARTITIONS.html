5.175 DBA_MINING_MODEL_PARTITIONS

DBA_MINING_MODEL_PARTITIONS describes all the model partitions accessible to the system. The columns in this view are same as those in ALL_MINING_MODEL_PARTITIONS.

Note:

This view is available starting with Oracle Database 12c Release 2 (12.2.0.1).
