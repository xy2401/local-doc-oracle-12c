Description of the illustration specifyMethodDetails.gif

This windows requests that you specify method and security details.On the left, a text box shows available methods with the following checked: CLRLibrary1, and under that, Class1, and then UInt32 GetDeptNo(UInt32).On the right is the title Methods Details, followed by Schema: with a drop-down list showing Scott as the selected schema. After that, is a text field for Database method that shows GETDEPTNO.After that, there is a drop-down list to set Security. The default level, Safe is entered.After that, there is a button labeled Parameter Type Mapping... which goes to that screen.There are buttons to go Back, Next, Finish, and Cancel.
