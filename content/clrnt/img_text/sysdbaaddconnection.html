Description of the illustration SYSDBAaddConnection.gif

This is the Add Connection window, with the Connection Details tab active and the Apply Filters tab inactive. The Connection Details tab shows the Data Source Name drop-down list, with Oracle selected. There are options to Use Windows integrated authentication or Use a specific user name and password, which is selected, with sys and *** in the fields a respectively.

There is a check box allowing you to save the password.

There is a drop-down list for Role, which shows SYSDBA grayed out. The connection name shows scott.Oracle.

There are the following buttons: OK, Cancel, and Help.
