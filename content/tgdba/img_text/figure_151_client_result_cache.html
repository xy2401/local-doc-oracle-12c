Description of the illustration client_result_cache.gif

This graphic shows a client process with two sessions. Each session executes the same SQL statement: SELECT department_id FROM departments. Each session retrieves the result set from the client result cache. The result set contains the values 10,20,30,40, and so on. A vertical dotted line divides this part of the diagram, labeled Client, from the part labeled Server. A database icon is shown interacting with the cache, with the label "Keeps Consistent."

End of image description.
