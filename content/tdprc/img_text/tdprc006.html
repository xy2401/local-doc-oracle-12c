Description of the illustration tdprc006.gif

The illustration shows a two node Oracle RAC database. The database instances communicate with each other. Each node also has an Oracle ASM instance that communicates with the Oracle ASM instance on the other node. These two nodes access the same ASM Clustered Pool of Storage.

The ASM Clustered Pool of Storage contains two disk groups. Disk Group A contains 3 disks and Disk Group B contains 2 disks.

End of description.
