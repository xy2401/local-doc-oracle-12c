This figures shows two of the configurations that are possible with Oracle Flex ASM. In the configuration labeled A there is both an Oracle Database instance and an Oracle ASM instance on the same Hub node. This demonstrates a local Oracle ASM client instance.

Configurations B and C show an Oracle Flex ASM client configuration. The database instance and Oracle ASM instance are on different Hub nodes in the cluster. Both the database instance and the Oracle ASM instance are in the same cluster. The database instances perform block I/O operations directly to Oracle ASM disks, and access Oracle ASM remotely for metadata. Some nodes in an Oracle Flex ASM configuration will have both an Oracle Database instance and an Oracle ASM instance, as shown in configuration A.
