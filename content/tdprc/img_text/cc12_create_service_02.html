Description of the illustration cc12_create_service_02.gif

The screenshot shows the High Availability Configuration portion of the Create Service page. The previous screenshot description describes the top section of this screenshot.

The first option to select is Cardinality, which can be either UNIFORM or SINGLETON. In the screenshot, the UNIFORM option is selected. The next field is a list for selecting the server pool in which the service runs. Following the list of server pools are hyperlinks for Edit and Add Server Pool. In the screenshot, the service is associated with the ora.backoffice server pool.

Underneath the Server Pool list is the text "UNIFORM services will run on all active servers of the server pool. SINGLETON services run on one of the active servers."

End of description.
